# Linux Daily Commands & Cheatsheet

## Overview

This repository is dedicated to documenting Linux commands and small
cheatsheets learned during daily practice.

The goal of this project is to build a strong foundation in Linux while
preparing for DevOps and cloud-related work. Each day new commands,
examples, or small notes will be added so the repository grows into a
complete Linux reference over time.

## What This Repository Contains

-   Daily Linux command cheatsheets
-   Basic to advanced Linux commands
-   Short explanations of commands
-   Practical examples used in the terminal
-   Personal learning notes

Each file added to this repository represents commands learned or
practiced on that day.

## Learning Approach

The learning process followed in this repository:

1.  Learn new Linux commands
2.  Practice them in the terminal (WSL / Ubuntu)
3.  Document them in a cheatsheet
4.  Upload the notes to this repository

This creates a structured and trackable Linux learning journey.

## Daily Updates

New cheatsheets or command notes will be uploaded regularly.\
These updates may include:

-   File and directory management commands
-   User and permission management
-   Process monitoring
-   Package management
-   Networking commands
-   Docker related Linux usage
-   DevOps related Linux practices

## Future Scope

This repository will gradually expand to include:

-   Advanced Linux administration commands
-   Shell scripting basics
-   Linux for DevOps engineers
-   Docker and container related commands
-   Kubernetes related Linux usage
-   Troubleshooting commands used in production environments
-   Real-world command examples used in DevOps workflows

Over time this repository will become a complete **Linux reference and
revision guide**.

## Purpose

The purpose of maintaining this repository is:

-   To track daily progress in Linux learning
-   To create a personal Linux command library
-   To help revise commands quickly
-   To build consistency in DevOps preparation
